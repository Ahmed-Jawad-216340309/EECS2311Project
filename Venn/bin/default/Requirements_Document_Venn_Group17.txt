1.	System Purpose
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	The purpose of the Venn Diagram Visual Tool is to allow users to organise their information visually into three categories, 
	consisting of sections of two circles and a section where the two circles overlap another. 
	Users are able to see the relationships between the sets of imformation and identify similarities and differences.
	All essential features of the system are described in this document.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


2.	System Features
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	- User is able to customize their venn diagram to their preference

		- Customizable color of each circle
		- Customizable size of the diagram
		- Customizable title of the visual diagram


	- User can add multiple text elements to the three segments of the venn diagram

		- Text elements can be added via dropdown menu
		- Text elements are able to be dragged and dropped to preferred diagram section.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


3.	Acceptance Requirements
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	The Acceptance Requirements met by the Venn Diagram Visual Tool.


	- User interface includes a venn diagram visual with a title above it, and subtitles above each circle.

	- Titles are customizable to user.
	
	- Venn Diagram allows user to add Text Elements into three seperate conjunctions.

	- Text elements are fully enclosed within the circles
	
	- The user interface includes a menubar at the top.
	
	- The menubar will have a dropdown menu which will include the menus:

		- 'Customize Circles'

			- Containing options for customizing the colors, borders, and size of circles.

				- Color option gives user ability to change either the left or right circle of the diagram.

			    	- Size option gives user options to set size of circles. 

				- Border option will allow user to customize the circles borders width and color.

		- 'Add Text Box'

			- Upon selection a new text box will be added to the right side of the user interface which will then
			   be available for dragging into the diagram sets and renaming.

			- The contents of any text box can be renamed after clicking it. The text boxes can also be dragged
			   into either circles or their conjunction.

		- 'Exit'
			- Will close the user interface upon selection.


	- Text element is held in the section it is dropped into by user.
  
	- Text elements exist in a list on the right side of the interface prior to dragging into diagram.

	- The different circles have same radius

	- The circle have an upper bound radius (due to screen size)

4.	Use Cases
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	The steps involved to perform the implementation of a user-personalized venn diagram are as listed below.
	
	
	- Changing Circles Colors
		1. Select 'Customize Circles' menu button located in the top left of the window.
		2. Select 'Color' option from the drop down menu.
		3. Select preferable color from color palet.

	- Setting Size of Circles using Preset Sizes
		1. Select 'Customize Circles' menu button located in the top left of the window.
		2. Select 'Size' option from the drop down menu.
		3. Select one of the three given options: 'small', 'medium', or 'large'.

	- Setting Custom Size of Circles
		1. Select 'Customize Circles' menu button located in the top left of the window.
		2. Select 'Size' option from the drop down menu.
		3. Enter pixel value into 'Custom Size' text field.

	- Change Thickness of Circle Border
		1. Select 'Customize Circles' menu button located in the top left of the window.
		2. Select 'Border' option from the drop down menu.
		3. Select 'Thickness' option.
		4. Select one of the border thickness options given.

	- Change Color of Circle Border
		1. Select 'Customize Circles' menu button located in the top left of the window.
		2. Select 'Border' option from the drop down menu.
		3. Select 'Color' option.
		4. Select preferable color from color palet.

	- Add Text Box to Universal List (right side of the window)
		1. Select 'Add Text Box' menu button located in the top left of the window.
		2. Text Box is added to Universal List with default text.

	- Changing Text in a Text Box
		1. Select Text Box with cursor.
		2. Modify Text to user preferance.

	- Move Text Box to a Section of the Venn Diagram
		1. Move mouse cursor above text field.
		2. Click and drag the text field to a section of the diagram.
		3. Drop the text field.
